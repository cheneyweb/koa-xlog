# koa-xauth
Node后端微服务框架，基于koa-xlog中间件，异步日志

[传送门：XServer官网文档](http://xserver.top)

框架目录结构
>
    ├── app.js
    ├── config
    │   ├── default.json
    │   ├── develop.json
    │   └── production.json
    ├── node_modules
    ├── package.json
    └── xlog_modules
        └── koa-xlog

快速上手
>
    1、const xauth = require('xlog')
    2、app.use(xlog())
